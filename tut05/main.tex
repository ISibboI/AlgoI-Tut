\begin{frame}{Das Problem}
Gegeben eine Folge von $n$ Elementen $\alpha_1, ..., \alpha_n$.

Finde eine Permutation $\sigma$, sodass gilt:

$\forall i \in \{1, \dots, n - 1\} : \alpha_{\sigma(i)} \leq \alpha_{\sigma(i + 1)}$
\end{frame}
